# Weex template based on yanxuan-weex-demo

## Usage

```
$ npm install
```

## Web

``` 
$ npm run build 
```

```
$ npm run dev & npm run serve 
```

## Ios

``` 
$ weex platform add ios
cd */platform/ios
$ pod install
```

```
$ weex run ios
```

## Android

``` 
$ weex platform add android
```
```
$ weex run android
```
