# WEEX TEMPLATE BASED ON yanxuan-weex-demo

## USAGE

```
$ npm install
```

## Web

``` 
$ npm run build 
```

```
$ npm run dev & npm run serve 
```

## Ios

``` 
$ weexpack platform add ios
cd */platform/ios
$ pod install
```

```
$ weex run ios
```

## Android

``` 
$ weexpack platform add android
```
```
$ weex run android
```
