# Usage
``` 
$ npm install
```

## Web

``` 
$ npm run build 
```

```
$ npm run dev & npm run serve 
```

## Debug

``` 
$ weex debug App.vue 
```
## Ios

``` 
$ weex platform add ios
cd /platform/ios
$ pod install
```

```
$ cd ../..
$ weex run ios
```

## Android

``` 
$ weex platform add android
```

```
$ weex run android
```
